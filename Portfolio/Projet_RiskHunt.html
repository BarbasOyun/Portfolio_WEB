Présentation du projet

Riskhunt est le projet que j'ai réalisé en lors de mon alternance en 3ème année à e-artsup l'objectif est de créer un outil pour des formateurs de professionnel.
Le projet est un éditeur de chasse aux risques sur des images 360°,
chaque utilisateur (formateurs) peut créer un compte et plus ou moins de scénarios en fonction du niveau de l'abonnement de son compte.
Ils peuvent ensuite partager leurs scénarios avec n’importe qui (les apprenants).

Scénarios

Création de scénario :

Le jeu permet de créer et/ou jouer des scénarios, tous le monde peut jouer des scénario sans se connecter
mais les formateurs ont besion de se connecter pour créer des scénarios.
Chaques scénario peut avoir plusieurs niveaux,
lors de la création de niveau ont doit d'abhord séléctionner une image en 360° sur la quelle la chasse aux risques va se dérouler,
ont peut ensuite placer un nombre de points limiter qui seront soit un text ou un quiz pour donner plus d'information sur le rique trouver.
*GIFS*

Stockage Scénario :

Les scénario une fois sauvegarder sont stocker sous la forme d'un dossier qui contient une image et un fichier binaire qui contient les informations des points
pour chaque niveau.
Dans les premières versions ce dossier était stocker localement sur l'ordinateur dans la version finale les scénarios sont stocker dans un server.
*Image save*

Récupération Scénarios :

Le jeu à besoin de récuperer un dossier de scénario qui peut venir d'un server ou de l'ordinateur, il peut ensuite convertir ce dossier en une classe
qui contient toutes les information pour instantier le scénario
*Image Load*
