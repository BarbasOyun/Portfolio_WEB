Présentation du projet

Riskhunt est le projet que j'ai réalisé en lors de mon alternance en 3ème année à e-artsup l'objectif est de créer un outil pour des formateurs de professionnel.
Le projet est un éditeur de chasse aux risques sur des images 360°,
chaque utilisateur (formateur) peut créer un compte et plus ou moins de scénarios en fonction du niveau de l'abonnement de son compte.
Ils peuvent ensuite partager leurs scénarios avec n’importe qui (les apprenants).

Scénarios

Création de scénario :

Le jeu permet de créer et/ou jouer des scénarios, tout le monde peut jouer des scénarios sans se connecter
mais les formateurs ont besoin de se connecter pour créer des scénarios.

Chaque scénario peut avoir plusieurs niveaux,
lors de la création du niveau on doit d'abhord sélectionner une image en 360° sur laquelle la chasse aux risques va se dérouler,
On peut ensuite placer un nombre de points limiter qui seront soit un texte soit un quiz pour donner plus d'informations sur le risque trouver.
*GIFS*

Stockage Scénario :

Les scénarios une fois sauvegardés sont stockés sous la forme d'un dossier qui contient une image et un fichier binaire qui contient les informations des points
pour chaque niveau.
Dans les premières versions ce dossier était stocké localement sur l'ordinateur dans la version finale les scénarios sont stockés dans un serveur.
*Image save*

Récupération Scénarios :

Le jeu à besoin de récuperer un dossier de scénario qui peut venir d'un server ou de l'ordinateur, il peut ensuite convertir ce dossier en une classe
qui contient toutes les informations pour instancier le scénario
*Image Load*

Le projet comporte un system de compte utilisateur avec une création de compte login/logout et suppression du compte.
La création du compte est requise pour créer des scénarios et les enregistrer.
Chaque compte peut enregister un certain nombre de scénarios.
