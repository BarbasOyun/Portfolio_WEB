ToyBox Warfare

Présentation du projet

Réaliser en groupe. Ce projet est un tower defence avec plusieurs types de tourelles que l'ont peut amélioré, le joueur peut également se transformer en tourelle, il est limité par un nombre de munitions, les ennemis peuvent lâcher des munitions après leurs mort, il y'a plusieurs type d'ennemis, plus ont va loin dans les vagues plus des ennemis puissant apparaîtront.

J'ai personnellement réaliser : système de tourelles, toutes les tourelles, placement des ponts, forme de tourelle du joueur.

Projet

Toutes les fonctionnalitée :

Toutes les tourelles ont des stats différentes : dégats, vitesse d'attaque, portée, coût. Le joueur peut placer plusieurs type de tourelles : tourelle basique(tirs un projectile droit), canon(zone de dégats), tourelle à bulle(ralenti les ennemies), tourelle piège(place un piège sur la route), boîte à musique(augmente les dégats des tourelles autour). Le joueur peut aussi améliorer les tourelles, pour quelles gagne des stats et évolue graphiquement. Le joueur peut placer des ponts sur la route car elle est bloqué pendant les vagues

Technique :

J'ai créer une class de base pour toutes les tourelles qui permet de les amélioré. Toutes les tourelles une fois placer rentre dans un état de construction ou elle ne peuvent rien faire, ont peut définir leurs prochain état au passage à l'état de construction. le background est une tilemap différente de la route se qui permet d'activé ou de désactivé le collider sur la tilemap de route pour empêcher le joueur de passer sur la route pendant les vagues
